lineboxintersect

synopsis

find intersections of lines with a box

syntax

    [x,y]=LINEBOXINTERSECT(lineparms,box)

description

[x,y]=LINEBOXINTERSECT(lineparms,box) lineparms is a nx2 matrix of theta/rho pairs that define a line. Box is a vector [xleft xright ybottom ytop]. The function returns the two intersection points of each line with the box (i.e. x and y are nx2 matrices) or NaN if the line does not intersect the box.

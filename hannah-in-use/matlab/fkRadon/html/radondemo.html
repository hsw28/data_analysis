radondemo

synopsis

demonstrates the radon transform

syntax

    RADONDEMO
    RADONDEMO(img)
    RADONDEMO(img,method)

description

RADONDEMO computes and shows the radon transform of an image and lets the user explore the transformation between image-space and radon-space.

RADONDEMO(img) uses the given image.

RADONDEMO(img,method) specifies custom method. Valid values are: integral, sum, mean, product, logsum (default='sum')
